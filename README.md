# BigInteger.kt

## Features
- [x] Addition
- [ ] Subtraction
- [ ] Multiplication
- [ ] Division
- [ ] Modulo
- [ ] Power
- [ ] Positive
- [ ] Negative

## Usage
```kotlin
val a = BigInteger.of(10_000_000_000)
val b = BigInteger.of(20_000_000_000)
val c = a + b
println(c) // 30000000000
```
